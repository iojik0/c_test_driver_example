c_test_driver_example

Простая программа на языке C, которая находит индексы, сумма соседних элементов которых максимальна.

Описание задачи

Программа находит индексы элементов массива (кроме первого и последнего), у которых сумма двух соседних элементов максимальна.

Для массива целых чисел необходимо найти все внутренние элементы (не первый и не последний), сумма значений соседних элементов которых максимальна.

Пример

Массив: [1, 2, 3, 4, 5]

Индекс 1: соседи 1 + 3 = 4
Индекс 2: соседи 2 + 4 = 6
Индекс 3: соседи 3 + 5 = 8 ← максимальная сумма

Результат: индекс 3

Как работает программа
Пользователь вводит размер массива (от 4 до 20)
Вводит элементы массива (от 0 до 1000)
Программа вычисляет для каждого внутреннего элемента сумму соседей
Находит максимальную сумму
Выводит все индексы с максимальной суммой соседей

Тестирование

Проект включает встроенный тестовый драйвер с 7 тестовыми случаями:

№	Название теста		Входные данные	Ожидаемый результат
1	Базовый случай		{1,2,3,4,5}	{3}
2	Несколько индексов	{10,1,10,1,10}	{1,3}
3	Все суммы равны		{5,0,5,0,5}	{1,3}
4	Максимум в начале	{1,100,1,0,1}	{2}
5	Максимум в конце	{1,0,1,100,1}	{2}
6	Все элементы равны	{7,7,7,7,7}	{1,2,3}
7	Минимальная длина	{1,2,3,4}	{2}

Тестовый драйвер запускается автоматически при старте программы.
